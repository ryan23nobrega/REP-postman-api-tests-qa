# 🧪 Postman API Tests - Fake Store

Projeto de testes de API utilizando Postman para simular cenários reais de QA.

## 🔧 Tecnologias utilizadas
- Postman
- FakeStore API

## 📌 Testes implementados

### GET /products
- Status code 200
- Tempo de resposta
- Validação de array
- Validação de conteúdo

### GET /products/{id}
- Validação de ID
- Validação de estrutura
- Testes dinâmicos com variáveis

### POST /products
- Criação de produto
- Validação de resposta
- Encadeamento de dados entre requests

## 🚀 Diferenciais
- Uso de environment variables
- Encadeamento entre requests (POST → GET)
- Validação automatizada com scripts

## 📂 Como rodar
1. Importar collection.json no Postman
2. Importar environment.json
3. Selecionar environment
4. Rodar a collection
